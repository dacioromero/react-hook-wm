# Changelog

## [0.1.2] - 2019-11-24
### Changed
- Reset `useHasPaid` on `requestId` change.
- Reset `useCounter` on `requestId` change.
- Default `useCounter` scale to `0`.

## [0.1.1] - 2019-11-24
### Changed
- Removed `use-immer` dependency.

## [0.1.0] - 2019-11-23
### Added
- `useStatus`
- `usePaymentPointer`
- `useRequestId`
- `useCounter`
- `useHasPaid`
